------Download Instructions------
1.) Click "Clone or Download" button from FlashcardApp repository.
2.) Click "Download ZIP"
3.) Save .zip file on your machine.
4.) Extract files

------Notes-------
1.) The enter key is enabled so you can press the enter key
on your keyboard rather than click "Submit Answer" every time.
2.) Press 'N' key at any point to change to a different card.

NOTE: There is no landing page, so the best place to start
is to open the addition.html file.
